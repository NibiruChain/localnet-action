# localnet-action

This action sets up a Nibiru localnet with toolings for testing.

This includes:
- nibiru chain
- faucet
- liquidation engine
- heart monitor
- bdjuno
- hasura
- big dipper
- price feeder

This is build using the latest image of each service.

You can run this using

```bash
    - uses: NibiruChain/run-chaosnet
```
